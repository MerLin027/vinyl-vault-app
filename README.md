## ✨ Features

*   **Sleek Dark Mode UI**: A premium, minimalist dark theme highlighted with soft teal accents, giving your records the visual stage they deserve.
*   **Smooth Navigation**: Custom `PageRouteBuilder` animations providing seamless fade-and-slide transitions across all screens.
*   **Curated Collection**: Browse exclusive vinyl records, categorized by genre (Rock, Jazz, Hip-Hop, Electronic, etc.).
*   **Responsive Design**: A fluid layout that adapts flawlessly to different screen sizes.
*   **Beautiful Typography**: Featuring *Tenor Sans* for an elegant logo wordmark and *Jost* for clean, modern interface text.

## 🚀 Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

*   Flutter SDK (3.x or higher)
*   Dart SDK
*   Android Studio / VS Code with Flutter extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/MerLin027/vinyl-vault-app.git
   ```

2. **Navigate to the app directory**
   ```bash
   cd vinyl-vault-app/my_app
   ```

3. **Install dependencies**
   ```bash
   flutter pub get
   ```

4. **Run the app**
   ```bash
   flutter run
   ```
